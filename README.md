# Love House는 가상으로 가구를 배치해보는 시뮬레이션 어플리케이션 입니다.

   * VR과 LeapMotion을 사용하여 손동작으로 이동, 가구배치가 가능합니다
   * 가구는 유저가 직접 Obj 형식의 파일을 구해야 합니다.
   * 가구 배치에 관해 save & load 가 가능하며 하나의 Obj 파일로 export가 가능합니다.
   * 배치된 모습을 jpg 파일로 저장이 가능합니다.
